## 수업 개요

본 수업은 기업가적 전환의 중요성에 대해 배우고 스스로 생각해보는 수업으로 가천대 코코네 스쿨만의 철학과 이념에 대해 배우는 전반부와 스타트업 중심의 기업가 정신의 핵심을 다루는 단계, 관련된 중요한 기타 주제들을 다루는 단계, 마지막으로 GCS를 중심으로 한 국내외 창업 생태계에 대해 고민해보는 후반부로 구성이 되어 있다. 각 단계에서 수강생들은 스스로 기업가정신에 대해 생각하고 토론을 통해 자신의 생각을 다른 사람들과 나누는 연습을 하게 될 것이다.

---

## 수업 목표

본 강의에서 수강자들은 아래의 사항들을 달성할 수 있어야 한다.

- 미래의 대학의 역할과 기업가적 전환의 필요성에 대해 이해한다.
- 다양한 종류의 기업가정신이 존재함을 이해하고 그 사례를 배운다.
- 스타트업에 필요한 기업가 정신이 무엇인지 이해한다.
- 인간 본성을 이해하는 일이 기업가정신에 왜 중요한지를 이해한다.
- 국내외 창업생태계에 대해 이해하고 나의 위치와 역할에 대해 이해한다.
- 자신만의 기업가 정신에 대해 깊이 고찰하고 다른 사람과 자신의 생각을 나눌 수 있다.

## 수업 방식

- 스타트업을 통해 기업가 정신, 기업가적 전환이라는 주제에 대해 스스로 생각하고 토론할 수 있도록 관련 강의와 함께 팀 토론대회를 병행할 예정

## 수업 참여

- 기업가 정신 수업은 출석과 토론 대회에 대한 참여도가 매우 중요한 수업입니다.

### 출석 규칙

- 결석 1회 9점 감점 ( 공결은 무감점 )
- 지각 1회 3점 감점 ( 지각 1회까지는 무감점 )
- 결석 3회부터는 토론대회 성적에 관계없이 A학점 부여 불가
- 결석 4회는 학칙에 의해 학점부여 불가(F학점)
- 온라인 강의의 경우에는 레포트 과제가 있습니다. 제출일 하루까지는 지각 처리하고 그 이상은 결석 처리합니다. 과제 점수는 따로 없으나 과제가 부실할 경우에는 지각 처리합니다.

### 세부 규칙

- 출석은 블루투스 방식으로 진행, 출석에 오류가 있는지 반드시 확인후 10분 안에 교수에게 이야기하여 정정할 것
- 수업시작 10분 후까지 출석으로 인정
- 수업시작 10분 후 도착한 학생은 1시간 이후 10분 휴식 때 교수에게 와서 지각 확인을 받을 수 있음.
- 수업시작 1시간 이후에 도착하면 도착하더라도 결석으로 간주.
- 따라서 조금 늦잠을 잤어도 한시간 안에 강의실로 오는게 성적에 유리함.

## 기업가정신 토론대회

- 100명을 3,4명씩 32개조로 나눈다.
- 32강 A그룹, 32강 B그룹 → 16강 → 8강 → 4강(준결승) → 결승 순으로 진행
- 스타트업, 기업가 정신과 관련된 토론주제를 미리 선정하여 공개하여 토론 준비를 하도록 할 예정

### 토론 주제 예시

"나는 무슨 일이든 반드시 된다는 믿음으로 시작한다." (故 정주영 회장) vs. "스타트업은 99%가 실패한다는 현실을 직시해야 한다." (에릭 리스, 『린 스타트업』) 스타트업을 시작하는 태도로서 두 관점은 서로 모순인가, 융화될 수 있는가?

### 라운드별 세부 진행 방식

경기는 **모두발언 → 질의응답 → 수강생 투표**순으로 진행된다.

|     라운드     | 출전 인원  | 모두발언 (팀당) | 질의응답 (팀당) | 투표·평가 | 총 소요 |
| :------------: | :--------: | :-------------: | :-------------: | :-------: | :-----: |
| 32강 (A/B그룹) | 팀대표 2명 |       2분       |       2분       |    2분    |  10분   |
|      16강      | 팀대표 2명 |       2분       |       2분       |    2분    |  10분   |
|      8강       | 팀대표 2명 |       3분       |       3분       |    2분    |  14분   |
|  4강(준결승)   |  팀 전원   |       3분       |       3분       |    2분    |  14분   |
|      결승      |  팀 전원   |       4분       |       4분       |    2분    |  18분   |

### 토론대회 점수 배분 (총 50점)

**1) 라운드 진출 점수 (팀 공통, 30점)**

|   단계    |  해당 팀  |       점수       |
| :-------: | :-------: | :--------------: |
| 32강 출전 |   전체    |       5점        |
| 16강 진출 | 상위 16팀 | +5점 (누적 10점) |
| 8강 진출  | 상위 8팀  | +5점 (누적 15점) |
| 4강 진출  | 상위 4팀  | +5점 (누적 20점) |
| 결승 진출 | 상위 2팀  | +5점 (누적 25점) |
|   우승    |    1팀    | +5점 (최대 30점) |

**2) 평가 적중 점수 (개인, 10점)**

경기 결과를 정확히 예측(투표)한 경우 라운드별로 아래 점수를 부여한다.
전 라운드 31경기 모두 투표 가능하며, 누적 합산 최대 10점 상한.

|  라운드  |  경기 수   | 적중 시 점수 |         라운드 만점         |
| :------: | :--------: | :----------: | :-------------------------: |
|   32강   |   16경기   | 경기당 0.2점 |            3.2점            |
|   16강   |   8경기    | 경기당 0.3점 |            2.4점            |
|   8강    |   4경기    | 경기당 0.5점 |            2.0점            |
|   4강    |   2경기    | 경기당 0.8점 |            1.6점            |
|   결승   |   1경기    | 경기당 1.0점 |            1.0점            |
| **합계** | **31경기** |              | **최대 10.2점 → 10점 상한** |

- 라운드가 올라갈수록 적중 점수가 높아져 후반 경기에 집중도를 높임
- 소수점 점수는 최종 합산 후 반올림하여 정수 처리

**3) 팀내 기여도 점수 (개인, 10점)**

경기 직후 팀원 전원이 설문에 참여하며, **팀원 N명에게 합계 100점을 배분**한다.

**산출 방식**

- 개인의 경기별 획득 점수를 전체 출전 경기에 걸쳐 누적 평균한다.
- 균등 기여 기준점 = 100 ÷ N (3인팀: 33.3점, 4인팀: 25.0점)
- 개인 평균 점수를 기준점과 비교하여 아래 비율로 10점 환산한다.

| 기준점 대비 개인 평균 비율 | 기여도 점수 |
| :------------------------: | :---------: |
|         140% 이상          |    10점     |
|   120% 이상 ~ 140% 미만    |     9점     |
|   100% 이상 ~ 120% 미만    |     8점     |
|    80% 이상 ~ 100% 미만    |     6점     |
|    60% 이상 ~ 80% 미만     |     4점     |
|          60% 미만          |     2점     |

- 설문 미참여 시 해당 경기 기여도는 0점으로 처리
- 자기 자신에 대한 배점은 설문에서 제외(타 팀원들만 평가)
